Changes in robotframework-pywinauto
==================================

This file lists the changes made to robotframework-pywinauto library.

Version 0.1.0 (2025-12-19)
=========================

Initial release
---------------

- **Core Functionality**
  - Basic application management (open, close, connect)
  - Window management (switch, minimize, maximize)
  - Control operations (click, type, get text)
  - Keyboard and mouse operations
  - Waiting mechanisms
  - Screenshot capture

- **Architecture**
  - Modular architecture following SeleniumLibrary patterns
  - Separate keyword modules by functionality
  - Support for multiple application instances
  - Context-aware keyword execution

- **Documentation**
  - Basic usage examples
  - Installation instructions
  - Keyword documentation

- **Tests**
  - Unit tests for core functionality
  - Acceptance tests for end-to-end scenarios
