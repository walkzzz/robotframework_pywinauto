# 修复Robot Framework文件错误

## 问题分析

### 1. examples/calculator.robot 文件
- 关键字 `Close All Applications` 应该使用小写加下划线 `close_all_applications`
- 所有关键字都应该遵循小写加下划线的命名规范

### 2. atest/acceptance/keywords/application.robot 文件
- 关键字 `Close All Applications` 应该使用小写加下划线 `close_all_applications`
- `Get Current Process ID` 和 `Get Current Application` 关键字应该使用小写加下划线命名
- `Evaluate` 关键字的使用存在问题，`process_id` 是属性不是方法
- `Get Current Application` 关键字实现有误，应该使用正确的方式获取当前应用实例

## 修复方案

### 1. 修复 examples/calculator.robot
- 将 `Close All Applications` 改为 `close_all_applications`
- 确保所有关键字都使用小写加下划线命名

### 2. 修复 atest/acceptance/keywords/application.robot
- 将 `Close All Applications` 改为 `close_all_applications`
- 将 `Get Current Process ID` 改为 `get_current_process_id`
- 将 `Get Current Application` 改为 `get_current_application`
- 修复 `get_current_process_id` 关键字的实现，使用正确的方式获取进程ID
- 修复 `get_current_application` 关键字的实现
